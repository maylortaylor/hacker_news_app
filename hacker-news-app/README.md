This is the overview of the folder structure of the entire app.

The **ClientApp** folder is where the full Angular 15 app is located.

│   │   |   |   |   ├── models
│   │   |   |   |   ├── pipes
│   │   |   |   |   ├── services

│   │   |   ├── app.component.html
│   │   |   ├── app.component.ts
│   │   |   ├── app.module.ts
│   │   |   ├── app.server.module.ts
│   │   ├── assets
│   │   ├── environments
│   │   ├── index.html
│   │   ├── main.ts
